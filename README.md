# Edu-Editor

**Edu-Editor is a basic medium/notion like rich text editor based on Slate.js framework.**

![Edu-Editor](https://user-images.githubusercontent.com/13861835/176963234-ab289945-eab3-4164-9b44-b461038a0255.gif)

## Basic Features
- blocks
- slash command
- paragraph
- headings
- lists
- quote
